# Language & Travel Planner (Client)

## Descripción
Aplicación web para ayudar a personas que viajan a otros países y quieren aprender frases básicas del idioma del destino.

Permite guardar destinos, ver frases útiles y practicar vocabulario básico.

## Tecnologías
- React
- TypeScript
- Vite
- LocalStorage

## Funcionalidades
- Lista de frases útiles por idioma
- Guardar destinos de viaje
- Quiz básico de vocabulario
- Interfaz simple tipo panel

## Objetivo
Facilitar el aprendizaje de frases básicas útiles durante viajes, todo en una aplicación sencilla.

## Nota
Los datos se guardan en LocalStorage, no se usa base de datos.
