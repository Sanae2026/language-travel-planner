# Metodologías de desarrollo

## Agile
Agile es una forma de trabajar en la que se construye software poco a poco, adaptándose a cambios.

## Scrum
Scrum divide el trabajo en sprints cortos con roles definidos.

## Kanban
Kanban usa columnas para organizar tareas (Todo, Doing, Done).

## Diferencias
Scrum es estructurado, Kanban es más flexible.

## Uso
Scrum para equipos grandes, Kanban para proyectos simples.
