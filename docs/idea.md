# Language & Travel Planner

## Problema
Muchas personas viajan a otros países sin conocer el idioma local, lo que dificulta situaciones básicas como pedir comida, moverse por la ciudad o comunicarse en hoteles.

## Usuario objetivo
Viajeros y estudiantes que quieren aprender frases básicas del idioma del país al que van a viajar.

## Funcionalidades principales
- Lista de frases útiles organizadas por idioma y situación (restaurante, transporte, hotel)
- Guardar destinos o países que el usuario quiere visitar
- Mini quiz para practicar vocabulario básico

## Funcionalidades opcionales
- Audio de pronunciación de frases
- Sistema de favoritos
- Modo offline usando LocalStorage

## Posibles mejoras futuras
- Añadir más idiomas y niveles de dificultad
- Sistema de usuario con inicio de sesión
- Recomendaciones automáticas según el destino
